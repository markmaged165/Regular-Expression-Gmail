Automated Information Extraction from Unstructured Text
This project demonstrates the use of Regular Expressions (Regex) and Python to parse messy, unstructured text paragraphs.

Key Features:
Extracts Full Names, Phone Numbers, DOBs, and Emails.

Converts raw text into a clean, structured Pandas DataFrame.

Handles various string patterns and formatting inconsistencies.

Technologies Used:
Python

Regex (re library)

Pandas
